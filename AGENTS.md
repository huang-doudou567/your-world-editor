## 快速理解

**你的世界编辑器** = 六层技术方案解决"人生认知"问题：记录层智能接入、情绪层自动分类、检索层对抗遗忘、洞察层跨越碎片、推演层向前看、分析层按标签深度分析。

## 核心规则

1. **读 SKILL.md**：完整系统逻辑、数据模型、Pipeline 都在里面
2. **数据目录**：所有人生系统文件在 `人生系统/` 目录下
3. **首次使用**：用户说「搭建」「启动」「初始化」→ 按 SKILL.md「首次使用：7步引导」执行
4. **日常使用**：用户说话 → 调用 `assess_record_worthiness()` 判断是否值得记录 → 按 SKILL.md 日常流转规则执行
5. **搜索**：用户说「上次」「之前」「找一下」→ 调用 `search_entries()` 加权搜索
6. **推演**：用户说「推演一下」→ 三条路径并行推演
7. **标签分析**：用户说「分析一下」「按标签看看」→ 触发标签维度分析

## 文件操作规则

- **写入**：直接写，事后可删，不阻塞节奏
- **流水账**：新条目加最上面（倒序），格式 `YYYY-MM-DD · [标签] 一句话标题`
- **情绪分类**：每条流水账自动标注 `emotion` 字段（🎨彩色 / 💡明亮 / 🌑黑暗）
- **模式匹配**：命中已有条目 → 案例日志加一行；新规律 → 建新条目
- **推演**：三条路径推演各自写入独立文件
- **Merge**：改变判断框架/自我认知 → 融入「我的故事」对应章节

## 自动化（手动触发）

Codex 沙箱环境无持久化定时任务，用户可手动触发：
- 周复盘：「手动周复盘」「帮我 review 这周」
- 决策回看：「回看一下三个月前的决策」
- 系统评估：「系统评估」「效果怎么样」
- 标签分析：「按标签看看」「分析一下」

## 沙箱环境注意事项

- 文件操作在沙箱内，退出后需导出
- 建议定期将 `人生系统/` 目录导出备份
- Python 脚本需在沙箱内运行：`python scripts/corpus/event_intake.py`

## 隐私

- 所有文件在沙箱内，用户完全控制
- 敏感人物用代号（M / J / 朋友A）
- 不想系统看到的内容，不记进系统
