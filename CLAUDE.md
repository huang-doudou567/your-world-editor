# 你的世界编辑器 · Claude Code 入口

> 这是我的自我认知操作系统。读完本文件 + SKILL.md，你就知道这个系统怎么用。

## 快速理解

**你的世界编辑器** = 三层技术方案解决"人生回溯"问题：

| 层 | 解决什么 | 模块 |
|---|---|---|
| **记录层** | 智能判断什么值得记，降低记录摩擦 | `scripts/corpus/event_intake.py` |
| **检索层** | 模糊记忆也能搜回来 | `scripts/corpus/store.py → search_entries()` |
| **洞察层** | 从碎片里自动发现"你又开始了" | `scripts/corpus/pattern_cluster.py` |

## 核心规则

1. **读 SKILL.md**：完整系统逻辑、数据模型、Pipeline、工具调用方式都在里面
2. **数据目录**：所有人生系统文件在 `人生系统/` 目录下
3. **首次使用**：用户说「搭建」「启动」「初始化」→ 按 SKILL.md「首次使用：7步引导」执行
4. **日常使用**：用户说话 → 调用 `assess_record_worthiness()` 判断是否值得记录 → 按 SKILL.md 日常流转规则执行
5. **搜索**：用户说「上次」「之前」「找一下」→ 调用 `search_entries()` 加权搜索

## 文件操作规则

- **写入**：直接写，事后可删，不阻塞节奏
- **流水账**：新条目加最上面（倒序），格式 `YYYY-MM-DD · [标签] 一句话标题`
- **模式匹配**：命中已有条目 → 案例日志加一行；新规律 → 建新条目
- **Merge**：改变判断框架/自我认知 → 融入「我的故事」对应章节

## 自动化（手动触发）

Claude Code 无内置定时任务，用户可手动触发：
- 周复盘：「手动周复盘」「帮我 review 这周」
- 决策回看：「回看一下三个月前的决策」
- 系统评估：「系统评估」「效果怎么样」
