# 你的世界编辑器

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/huang-doudou567/your-world-editor)

> 一个 AI 驱动的自我认知操作系统
>
> 记录层降低摩擦 · 检索层对抗遗忘 · 洞察层跨越碎片 · 推演层向前看 · 情绪层增加情感维度

---

## 解决什么问题

人的自我认知依赖记忆，但记忆是衰减的、碎片的、情绪偏差的。三个月前为什么做了那个决定，当时怎么想的，这已经是第几次了——说不清楚。这个信息不对称导致：做决策时调用不了完整信息源；反复踩同一个坑而不自知。同时，人在面临重大选择时缺少"向前看"的工具——迷茫时不知道该往哪走，恐惧时不知道选了会不会很糟。

## 怎么解决

把"人生认知"拆成五个独立的技术问题：

| 层 | 模块 | 做什么 | 实现 |
|---|---|---|---|
| **记录层** | 智能事件接入 | 不靠触发词，自动判断什么值得记 | `scripts/corpus/event_intake.py` |
| **情绪层** | 语义情感分类 | 每条事件自动标注情感色调 | Agent 语义判断 → `emotion` 字段 |
| **检索层** | 多维度加权搜索 | 模糊记忆也能捞回来 | `scripts/corpus/store.py` |
| **洞察层** | 主题聚类+周期检测 | 从碎片里自动发现"你又开始了" | `scripts/corpus/pattern_cluster.py` |
| **推演层** | 人生路径推演 | 同时推演多条路径，看清每条路的终点 | `人生系统/决策推演/` |

五层之上有评估指标闭环：系统健康度 / 盲区改善度 / 叙事沉淀率。

## 文件结构体系

### 基础三层（向后看）

| 层 | 文件 | 比喻 | 装什么 |
|---|------|------|--------|
| **事件层** | `经历流水账.md` | 收件箱 | 发生的事，倒序，随手记。每条附带情绪标签 |
| **规律层** | `模式与盲点.md` | 雷达 | 反复出现的行为模式 / 盲区 / 情绪触发 |
| **叙事层** | `我的故事.md` | 图书馆 | 判断框架、价值观、人生叙事 |

### 情绪维度

每条事件自动分类到三个情绪窗格之一：🎨 彩色窗格 / 💡 明亮窗格 / 🌑 黑暗窗格。不评判好坏，只帮用户看见"我最近住在哪个窗格里"。

### 推演层（向前看）

`人生系统/决策推演/` —— 同时推演三条人生路径（当前/转型/理想），看清每条路的终点和代价。

## 多平台支持

| 平台 | 入口文件 | 使用方式 |
|------|---------|---------|
| **Claude Code** | `CLAUDE.md` + `SKILL.md`  | 放到项目目录，Claude Code 自动读取 |

## 快速开始

跟 AI 说：「搭建」「启动」「初始化」

引导流程：起昵称 → 选文件位置 → 建核心文件 → 投喂初始资料 → 书摘联动 → 试一句流水账 → 收尾

## 日常使用

说话就行，不用记命令。

- 📝 记流水账：「记一下」「今天」—— 或者直接说，AI 会自动判断是否值得记录
- 🎨 情绪窗格：「放彩色窗格」「放黑暗窗格」—— 每条事件自动标注情感色调
- 🔍 检索回忆：「上次跟XX吵架那次我怎么想的来着」
- 🎯 辅助决策：「要不要」「纠结」「帮我分析」
- 🧭 决策推演：「推演一下」「帮我推演几条路」—— 同时推演三条不同的人生路径
- 📚 沉淀叙事：「merge 一下」「值得记住」
- 🔍 周复盘：「周复盘」「照镜子」—— 含本周情绪分布统计
- 📊 系统评估：「系统评估」「效果怎么样」
- 💼 求职 · 💰 财务 · 🏥 健康 · ❤️ 爱情 · 🌙 梦境
